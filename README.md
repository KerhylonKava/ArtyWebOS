# ArtyWebOS

This is a Web-based Operating system made for <a href="https://stardance.hackclub.com/">Stardance!</a>

It was made loosely following this guide: <a href="https://jams.hackclub.com/batch/webOS">Create Your Personal Web-based Operating System</a>


I wrote the code, with some help from my dad, and some borrowing from the code of my personal website (which I also wrote).

## Features

- A draggable, close-able, and open-able welcome window, as well as an Icebreaker "app" that has a ~~daily~~ icebreaker question.

- A working date+time display in the top right corner\
